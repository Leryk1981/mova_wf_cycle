# skill.mova_ai_bootstrap_generate_basic

Базовий жовтий skill для генерації MOVA AI bootstrap-пакета.
- Вхід: `env.mova_ai_bootstrap_generate_v1` (опис цільової моделі/середовища).
- Вихід: `ds.mova_ai_bootstrap_pack_v1` (статичний JSON з інструкціями, правилами, рецептами, прикладами).
- Не викликає зовнішні моделі, не змінює репозиторій; працює зі статичним ресурсним JSON.
