# MOVA overlays for `skill.runtime_binding_code_scaffolder_basic`

This folder contains local MOVA definitions for the skill:

- `ds/` – local data schemas:
  - `ds.runtime_binding_scaffold_request_v1.schema.json` – input schema describing skill/envelope requirements and runtime target
  - `ds.runtime_binding_scaffold_result_v1.schema.json` – output schema with code skeleton files and optional binding JSON

- `env/` – local envelopes:
  - `env.runtime_binding_scaffold_run_v1.schema.json` – envelope for running the runtime binding scaffolder skill
