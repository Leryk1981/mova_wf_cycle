# Skills (yellow layer)

Each subfolder here will define a single skill, for example:

- `skill.file_cleanup`

- `skill.obsidian_notes`

- `skill.social_forms`

- `skill.youtube_pipeline`

A skill will follow this canonical structure:

- `manifest.skill.json`

- `mova/ds` and `mova/env` – local overlays over the red core

- `cases/` – example inputs/outputs

- `episodes/` – real execution traces

- `impl/` – prompts, code, adapters

In this bootstrap step, do NOT create any concrete skills yet.

Only keep this README.
