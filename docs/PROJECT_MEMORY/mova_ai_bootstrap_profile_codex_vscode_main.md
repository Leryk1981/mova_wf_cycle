# MOVA AI bootstrap — codex_vscode_main

codex_vscode_main — головний Codex-подібний агент у VS Code/MOVA Workbench, який виконує роль MOVA-архітектора/екзек’ютора.

- Envelope: `lab/examples/env.mova_ai_bootstrap_generate_v1.codex_vscode_main.json`
- Pack: `lab/examples/mova_ai_bootstrap_pack.codex_vscode_main.v1.json`
- Згенеровані профілі: `lab/generated/bootstrap/codex_vscode_main.system_prompt.txt`, `codex_vscode_main.assistant_guide.md`, `codex_vscode_main.checklist.txt`

Підключення у VS Code:
- У конфігурації агента (Codex/AI розширення) базові інструкції (system prompt) взяті з `codex_vscode_main.system_prompt.txt`.
- Guide/checklist використовуються як довідка/нагадування в окремих вкладках або нотатках.
